Projects


Track My Cruise

TrackMyCruise is application, which allows you to get retrospective marine weather data (wave height, wind, temperature, etc.) for couple of years's back in history, where you can keep the records in a form of cards. This application have been builded up in a team of 4 people and launched in just 2 weeks time. My main contribution was the back-end programmed on Ruby On Rails and the page with the cards, where the user can update a card on a modal page with picture preview option coded with JavaSctipt, HTML & CSS .I also created Cloudinary Web API in order to upload images.

Cooking Venue

Boat BNB

The application, was builded up in a team of 4 people, where my main contribution is the design part created on Figma , I did a lot of HTML & CSS ,I was involved with the back-end ( Ruby On Rails ) and also created Cloudinary Web API in order to have cool image gallery.
